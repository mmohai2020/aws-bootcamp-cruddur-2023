# aws-bootcamp-cruddur-2023

## Nonprofit Landing Page

A modern, responsive landing page for a West African nonprofit organization focused on poverty alleviation.

### Location
`/nonprofit-landing/`

### Features
- 6-slide hero banner with auto-rotation
- Donation system with multiple amounts
- Volunteer signup form
- Program showcase
- Impact section for photos/videos
- Fully responsive design
- Mobile-friendly navigation

### Quick Start
Open `/nonprofit-landing/index.html` in a web browser.

See `/nonprofit-landing/README.md` for detailed documentation.
